# 👨‍💻 Tech stack

Here's a brief high-level overview of the tech stack the World protocol uses:


- Virtual environement
  - WSL 2 (if using windows)
  - Docker Compose
  - Remote tools
  - GitPod, GoogleCollab


- Back-end
  - Solitity
  - Metamask / Ronin
  - Truffle  / Hardhat
  - Alchemy
  - OpenZeppelin 
  - ERC / EIP
  - TraderJoe, SushiSwaps (external scripts)

- Blockhains dev arch
  - Ethereum
  - Polygon
  - Avalanche
  - Solana
  - Bitcoin  

- Client
  - ReactJS
    - Snackbar
    - React-Router 
  - NextJS
  - Material-UI, Emotion
  - Styled-components
  - Redux
  - Web3
  - Ethers.js
  - Moralis.Web3

- Api
  - Node.js
  - Express

- Testing 
  - Ganache
  - Mocha
  - Chai
  - HRE
  - Enzyme
  - Foundry

- Deployment
 - Netlify
 - Vercel
 - Cloud broadcast
